Golang isn’t clear like Ruby or Python, but its clear enough low level languages as far. It’s OK if you learned as much as. Golang is function-oriented. But there is a way to use functions as like class methods.

Fast

Async

When I was writing async code in Python, I’ve been searching for async packages. Mailing, ram access, disk access.. You can stuck with that if you can’t find a good fit your project. Synchronous packages will block your async code. It was just like pain in the ass.

In Golang, you don’t need specific packages. You can fire functions asynchronously with go keyword.

// run sync
myFunc()
//run async
go myFunc()
